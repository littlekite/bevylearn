**bevylearn 一起学习bevy**
## 项目名称  绝地求生

#### 基本描述

1. 2D像素
2. 固定场景、摄像机位
3. 前期对战AI 后期添加多人对战
4. 仅支持 左右操作和发射子弹  没有上下移动
5. 有血条  中弹减少血量  血条为0 时默认失败

#### 中立区域

```seq
会随机生产AI 敌人 随机向上或者向下攻击
会掉落血包 吃到会回复一定量的血量
会掉落武器 吃到会更换当前武器 参考红斗罗
血包和武器可以被攻击，会被打爆
```
